    def hello(name):←仮引数name

１３．関数とは・関数を作る・関数の使い方

  • 関数
    関数とは、ある処理をまとめたプログラムの塊で、printも関数の1つ。本来、コンソールに文字を出力するためには色々なコードを書く必要があるが、printという関数のおかげでprintとだけ書けばいいようになっている。
  • 関数について
    printの他にもPythonにはいくつかの便利な関数が用意されており、それらを使うと様々な処理を簡単に行うことができる。また、関数は自分で作ることもできます。これらの関数を組み合わせることで、プログラムを効率的に作ることができる。
  • 関数を作る
    関数の定義
    def 関数名():
        実行する処理

    コード例
    def hello():
        print("Hello World")

    関数は「def 関数名():」のように定義する。def=define（読み＝デフ・ディファイン、意味＝定義する）
  • 関数の使い方
    def hello():
        print("Hello World")←この2行が関数の定義

    hello()←関数の呼び出し

    関数は定義しただけでは実行されないので、呼び出して実行する。関数名()のように() をつけて呼び出す。ただし、関数は定義した後でしか呼び出せないので気をつける。

１２．while文,無限ループ,break,continue

  • while文→
    x = 1←条件式で使う変数を定義
    whil x <=100:←条件式
        print(x)
        x += 1←この2行は条件式がTrueの間繰り返される処理を記述（xを出力することと、xの値が１ずつ増えていくこと）

    ＃出力結果
    1
    2
    ...
    99
    100
    繰り返し処理にはfor文以外にもwhile文というものがある。while文を用いると、「ある条件に当てはまる間、処理を繰り返す」といったことが可能になる。「while 条件式:」のように書く。while(読み＝ワイル、意味＝～する間)
  • 無限ループ→while文では、処理の最後に変数の値を更新し忘れると、条件が常にTrueになってしまうので、繰り返し処理が無限に行われる（無限ループ）。無限ループはコンピュータに異常な負荷をかけることになるので、確実に阻止しなければならない。繰り返し処理では、必ずどこかで条件がFalseになるように実装すること。
  • break
    numbers = [1,2,3,4,5,6]
    for number in numbers:
        print(number)
        if number ==3:
            break←xが3になった時点で繰り返し処理終了

    breakを用いると繰り返し処理を終了することができる。if文などの条件分岐と組み合わせて使う。while文でも同じように使うことができる。
  • continue
    numbers = [1,2,3,4,5,6]
    for number in numbers:
        if number % 2 == 0:
            continue←numberが2で割り切れる（偶数）とき、その周の繰り返しをスキップ
        print(number)

    繰り返し処理を終了するbreakと違い、continueはその周の処理だけをスキップすることができる。continueもif文などと組み合わせて利用する。while文でも同じように使うことができる。

１１．辞書の要素の(更新・追加・取り出し方)

  • 辞書の要素の更新→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    fruits["apple"] = 'green'←キー名がappleである要素の値を更新
    辞書名[キー名] = 値と書くことで要素の更新をすることができます。
  • 辞書の要素の追加→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    fruits["peach"] = 'pink'←キー名がpeachである要素の値を追加
    print(fruits)
    ＃結果出力
    {'peach' : 'pink','apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    「辞書名[新しいキー名] = 値」と書くことで辞書に新しい要素を追加することができる。ただし、辞書にすでにあるキー名を指定すると、値の追加ではなく更新になってしまうので注意。
  • 辞書の要素の取出し方→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    for fruit_name in fruits←キーが一つずつ取り出され、変数に代入される
        print(fruit_name + 'の色は' + fruits[fruit_name] 'です')←キー名を用いて値を取り出す

    リストと同じように、辞書もfor文を用いて要素を1つずつ取り出し、処理を行うことができる。「for 変数名 in 辞書:」と書くことで繰り返し処理をすることができる。ここで定義された変数にそれぞれの要素のキーが1つずつ代入される。要素の値はキーが代入された変数を用いて取り出せる。

１０．for文,辞書(書き方,要素の順番・取り出し方)

  • for文の書き方→
    foods = [pasta,curry,sushi]
    for food in foods:
        print("好きな食べ物は" + food + "です")
    ＃出力結果
    好きな食べ物はpastaです
    好きな食べ物はcurryです
    好きな食べ物はsushiです

    for文を使うと、リストの要素を順に取り出して処理を行うことができます。 下図のように「for 変数名 in リスト:」と書くことで、リストの要素の数だけ、処理を繰り返すことができる。
  • for文(繰り返し処理)→
    foods = [pasta,curry,sushi]
    for food in foods:
        print("好きな食べ物は" + food + "です")←for文の中身

    「for 変数名 in リスト:」とすると変数には、リストの要素が先頭から順に1つずつ入っていき、その上でfor文の中の処理が実行される。処理はリストの要素の数だけ繰り返し行われる（繰り返し処理）。また、変数名は自由ですが、リスト名の単数形にすることが慣習上多い。
  • 辞書とは→辞書は、リストと同じように複数のデータをまとめて管理するのに用いられる。リストとの違いは、個々の要素をインデックス番号ではなくキーと呼ばれる名前を付けて管理する点。辞書ではキーと値のペアが1つの要素となる。
  • 辞書の書き方→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    辞書は{キー1: 値1, キー2: 値2, …}のように作る。ほとんどの場合、キーには文字列が使われる。リストは要素を[]で囲んだが、辞書は{}で囲む。キーと値の間はコロン（:）、要素同士の間はコンマ（,）で区切ります。
  • 辞書の要素の順番→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    print(fruits)
    ＃出力結果
    {'banana' : 'yellow','grape' : 'purple','apple' : 'red'}←辞書内の要素には順序がないので定義したときの順番と異なる
  • 辞書の要素の取出し方→
    fruits = {'apple' : 'red','banana' : 'yellow','grape' : 'purple'}
    print('appleの色は' + fruits['apple'] + 'です')
    ＃出力結果
    appleの色はredです
    辞書の値を取り出すには、取り出したい値に対応する「キー」を用いて辞書名[キー]のように書く。

７．論理演算子(and,or),not(条件式)

  • and→
    論理演算子 - and
    True and True→True
    True and False→False
    False and True→False
    False and False→False

    time = 14
    if time > 10 and time < 18:
        print("就業時間です")

    ＃結果就業時間です
    「条件1も条件2も成り立つ」というような場合の条件式は「条件1 and 条件2」のようにかく。andを用いて複数の条件式を組み合わせるとすべての条件式がTrueの場合全体がTrueとなる。
  • or→
    論理演算子 - or
    True or True→True
    True or False→True
    False or True→False
    False and False→False

    time = 15
    if time == 10 or time == 15:
        print("おやつの時間です")

    ＃おやつの時間です
    「条件1か条件2が成り立つ」というような場合の条件式は「or」を用いて、「条件1 or 条件2」のように書きます。「or」を用いて複数の条件式を組み合わせると、複数の条件式のうち1つでもTrueであれば全体がTrueとなります。
  • not→
    time == 9
    if not time == 18:
        print("退社時間ではありません")

    ＃結果退社時間ではありません
    「not」を用いると、条件の否定をすることができます。「not 条件式」のようにすると、条件式が「True」であれば全体が「False」に、「False」であれば「True」になります。

６．if文と真偽値,比較演算子(<,<=,>,>=),else,elif

  • 真偽値→
    print(3 == 3)
    ＃結果Trueと表示
    print(3 == 5)
    ＃結果Falseと表示
    真偽値を扱うデータ型、「真偽値型」という比較演算子を用いた条件式の部分が、成り立つとき「True」、成り立たないとき「False」の2つがある。頭文字は大文字。True（読み=トゥルー、意味＝真）False（読み＝フォールス、意味＝偽）。
  • if文と真偽値→if文では条件式がTrueのときは処理が実行され、Falseのときは処理が実行されない。
  • 比較演算子(<,<=,>,>=)→
    比較演算子 - 大小を比べる
    x<y   ←xがyより小さいときTrue
    x<=y  ←xがyより小さいまたは等しいときTrue
    x>y   ←xがyより大きいときTrue
    x>=y  ←xがyより大きいまたは等しいときTrue

    比較演算子には、==や!=といった値の等しさを比較する記号以外に、値の大小を比較する記号がある。条件式が成り立てばTrue,成り立たなければFalseをそれぞれ返す。
  • else→
    score = 50
    if score == 100
        print("よくできました！")
    else:
        print("頑張りましょう")

    if文にelseを組み合わせることで「もし○○ならば△△を行う、そうでなければ□□を行う」という条件分岐ができるようになる。if文の条件がFalseのときelseの処理を実行する。else（読み＝エルス、意味＝そうでなければ・その他・別の）。
  • elif(1)→
    score = 70
    if score == 100
        print("よくできました！")
    elif score >= 70
        print("まずますです...")←実行される

    else:
        print("がんばりましょう！")

    if文で条件が成り立たなかった場合を複数定義したい場合は、elifを用いる。「もし○○なら△△を行う、××なら▼▼を行う、そうでなければ□□を行う。」elif=else if（読み＝エルイフ、意味＝もしそうでなければ）。
  • elif(2)→
    score = 100
    if score == 100:
        print("よくできました")　←最初の条件が成立したのでそれ以降の処理は実行されない

    elif score >= 60:
        print("まずますです...") ←条件がTrueでも処理が実行されない
    elif score >= 40:
    ...
    else:
    ...
    elifはいつでも書けるが、上から順に条件が成り立つか判断され、最初に条件が合致した部分の処理だけが行われる。

５．条件分岐,if文,if文条件式(比較演算子==,!=),インデント

  • 条件分岐→条件（テストの点数）によって処理（成績）を変えるというのが条件分岐。
  • if文→
    score=100
    if score == 100: ←条件式が成り立つ
    print("よくできました") ←条件が成り立つの時の処理
    if文を用いると、「もし○○ならば△△を行う」という条件分が行われる
  • if文の条件式→
    比較演算子 -　等しいか調べる
    x==y ←左右の値が等しいときに成り立つ
    x!=y ←左右の値が等しくない時に成り立つ

    score = 100
    if score = 100:
    条件式のなかでは、二つの値を比較する「比較演算子」がよく使われる。if文の条件部分はif条件式：のように書く。
  • インデント→
    score = 50
    if score == 100
        print("よくできました")
        print("次も頑張りましょう！”）

    ↑※インデントをそろえる。半角スペース４つ分
    ＃結果条件式が成り立たないため表示されない

    score = 50
    if score == 100
        print("よくできました")
    print("次も頑張りましょう！")

    ↑※インデントが無いためif分外とみなされる
    ＃結果次も頑張りましょう！と表示されてしまう。
    処理がif文内にあるかどうかはインデントによって判別される。条件が成立したときif文の中の処理が実行される。

２．変数の定義、規格、意義

  • 変数をイメージ→変数とは、データ（値）を入れておく箱のようなもの。この箱（変数）に名前（変数名）をつけることでその名前を用いて変数に値を入れたり変数から値を取り出したりできる。
  • 変数の定義→変数名="数値"で定義する。数値のみクォーテーションで囲む。プログラミングの＝は右辺を左辺に代入するという意味。
  • 変数の値を抽出→
    name="john"
    print(name)
    とすると変数nameの値"john"が出力。print("name") としてしまうと文字列"name"を出力する。
  • 変数名のつけ方→英単語を用いる。２語以上の場合は＿で区切る。数字で開始、ローマ字、日本語は推奨されていないもしくはできない。
  • 変数を使う意義１→
    ☆使用した場合☆
    apple    _count=3
    apple_price=100
    toral_price=apple_count* apple_price
    print(total_price)結果出力
    ☆使用しない場合☆
    total__price=3*100
    print(total_price)結果出力
    のようにデータに名前を付けることで扱っているデータの中身が何を表しているか明確になる。その結果コードが読みやすい。変数を使うメリット。
  • 変数を使う意義２→
    ☆使用した場合☆
    #正方形の面積を計算
    length=5
    area=length*length
    print(area)結果出力
    ☆使用しない場合☆
    #正方形の面積を計算
    area=5*5
    print(area)結果出力
    のように同じデータを繰り返し使用することができ、変数の値に変更があった場合変更する箇所が１箇所ですむ。

１．#,print(),文字列,数値

  • print("テキスト")→文字列。文字を出力する。シングル又はダブルクォーテーションで囲む必要。単語意味＝印刷する
  • #→行頭に書くことで行末までコメントとみなされる。読み＝パウンド。
  • print(4 + 3)→数値。計算結果が出力される。半角必須、スペース推奨。
  • print(+-*/%)→様々な計算。小数点も出力。%は割算で余りを出力する。
