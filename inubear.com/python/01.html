９．リスト(変数に代入,要素を取得・更新・追加)

  • リスト→
    リストの作り方
    [要素1,要素2,要素3]

    ＃文字列のリスト
    ["pasta","curry",sushi]
    ＃数値のリスト
    [1,3,5,8,13,21]
    ＃文字列と数値のリスト
    [apple,banana,200,300]
    複数のデータをまとめて管理するにはリストというものを用います。複数のデータをまとめて管理するにはリストというものを用います。リストは[要素1, 要素2, ...]のように作ります。リストに入っているそれぞれの値のことを要素と呼びます。リストを使うと、複数の文字列や複数の数値を1つのものとして管理することができます。
  • リストを変数に代入→
    foods = [pasta,curry,sushi]
    print(foods)
    ＃出力結果[pasta,curry,sushi]←リストがそのまま出力さえる
    リストも1つの値なので変数に代入することができます。このとき、リストを代入する変数名は慣習上複数形にすることが多いので、覚えておきましょう。
  • リストの要素を取得→
    foods = [pasta,curry,sushi]
               0     1     2  ←#インデックス番号
    print("好きな食べ物は" + foods[2] + "です”）
    #結果-好きな食べ物はsushiです

    リストの要素には、前から順に「0, 1, 2,・・・」と数字が割り振られている。これをインデックス番号という。インデックス番号は0から始まる点に注意。リストの各要素は、リスト[インデックス番号]とすることで取得することができる。
  • リストの要素を更新→
    foods = [pasta,curry,sushi]
    foods[1] = "pizza"
    print(foods)
    ＃出力結果
    [pasta,pizza,sushi]
    「リスト[インデックス番号] = 値」とすることで、リストの指定したインデックス番号の要素を更新することができる。
  • リストの要素を追加→
    foods = [pasta,curry,sushi]
    foods.append("pizza")
    ＃リストの末尾に要素を追加
    prit(foods)
    ＃出力結果
    [pasta,curry,sushi,pizza]
    「リスト.append(値)」とすることで、すでに定義されているリストの末尾に新たな要素を追加することができる。append(意味＝付加する、読み＝アペンド)。

    ...

７．論理演算子(and,or),not(条件式)

  • and→
    論理演算子 - and
    True and True→True
    True and False→False
    False and True→False
    False and False→False

    time = 14
    if time > 10 and time < 18:
        print("就業時間です")

    ＃結果就業時間です
    「条件1も条件2も成り立つ」というような場合の条件式は「条件1 and 条件2」のようにかく。andを用いて複数の条件式を組み合わせるとすべての条件式がTrueの場合全体がTrueとなる。
  • or→
    論理演算子 - or
    True or True→True
    True or False→True
    False or True→False
    False and False→False

    time = 15
    if time == 10 or time == 15:
        print("おやつの時間です")

    ＃おやつの時間です
    「条件1か条件2が成り立つ」というような場合の条件式は「or」を用いて、「条件1 or 条件2」のように書きます。「or」を用いて複数の条件式を組み合わせると、複数の条件式のうち1つでもTrueであれば全体がTrueとなります。
  • not→
    time == 9
    if not time == 18:
        print("退社時間ではありません")

    ＃結果退社時間ではありません
    「not」を用いると、条件の否定をすることができます。「not 条件式」のようにすると、条件式が「True」であれば全体が「False」に、「False」であれば「True」になります。

６．if文と真偽値,比較演算子(<,<=,>,>=),else,elif

  • 真偽値→
    print(3 == 3)
    ＃結果Trueと表示
    print(3 == 5)
    ＃結果Falseと表示
    真偽値を扱うデータ型、「真偽値型」という比較演算子を用いた条件式の部分が、成り立つとき「True」、成り立たないとき「False」の2つがある。頭文字は大文字。True（読み=トゥルー、意味＝真）False（読み＝フォールス、意味＝偽）。
  • if文と真偽値→if文では条件式がTrueのときは処理が実行され、Falseのときは処理が実行されない。
  • 比較演算子(<,<=,>,>=)→
    比較演算子 - 大小を比べる
    x<y   ←xがyより小さいときTrue
    x<=y  ←xがyより小さいまたは等しいときTrue
    x>y   ←xがyより大きいときTrue
    x>=y  ←xがyより大きいまたは等しいときTrue

    比較演算子には、==や!=といった値の等しさを比較する記号以外に、値の大小を比較する記号がある。条件式が成り立てばTrue,成り立たなければFalseをそれぞれ返す。
  • else→
    score = 50
    if score == 100
        print("よくできました！")
    else:
        print("頑張りましょう")

    if文にelseを組み合わせることで「もし○○ならば△△を行う、そうでなければ□□を行う」という条件分岐ができるようになる。if文の条件がFalseのときelseの処理を実行する。else（読み＝エルス、意味＝そうでなければ・その他・別の）。
  • elif(1)→
    score = 70
    if score == 100
        print("よくできました！")
    elif score >= 70
        print("まずますです...")←実行される

    else:
        print("がんばりましょう！")

    if文で条件が成り立たなかった場合を複数定義したい場合は、elifを用いる。「もし○○なら△△を行う、××なら▼▼を行う、そうでなければ□□を行う。」elif=else if（読み＝エルイフ、意味＝もしそうでなければ）。
  • elif(2)→
    score = 100
    if score == 100:
        print("よくできました")　←最初の条件が成立したのでそれ以降の処理は実行されない

    elif score >= 60:
        print("まずますです...") ←条件がTrueでも処理が実行されない
    elif score >= 40:
    ...
    else:
    ...
    elifはいつでも書けるが、上から順に条件が成り立つか判断され、最初に条件が合致した部分の処理だけが行われる。

５．条件分岐,if文,if文条件式(比較演算子==,!=),インデント

  • 条件分岐→条件（テストの点数）によって処理（成績）を変えるというのが条件分岐。
  • if文→
    score=100
    if score == 100: ←条件式が成り立つ
    print("よくできました") ←条件が成り立つの時の処理
    if文を用いると、「もし○○ならば△△を行う」という条件分が行われる
  • if文の条件式→
    比較演算子 -　等しいか調べる
    x==y ←左右の値が等しいときに成り立つ
    x!=y ←左右の値が等しくない時に成り立つ

    score = 100
    if score = 100:
    条件式のなかでは、二つの値を比較する「比較演算子」がよく使われる。if文の条件部分はif条件式：のように書く。
  • インデント→
    score = 50
    if score == 100
        print("よくできました")
        print("次も頑張りましょう！”）

    ↑※インデントをそろえる。半角スペース４つ分
    ＃結果条件式が成り立たないため表示されない

    score = 50
    if score == 100
        print("よくできました")
    print("次も頑張りましょう！")

    ↑※インデントが無いためif分外とみなされる
    ＃結果次も頑張りましょう！と表示されてしまう。
    処理がif文内にあるかどうかはインデントによって判別される。条件が成立したときif文の中の処理が実行される。

  • 文字列の連結→

    name="john"

    数値の計算で用いた＋記号は計算だけでなく文字列の連結も行うことができる。他、変数と文字列や変数と変洲も連結できる。
  • データ型→
    "hello python"-文字列型
    3-数値型
    文字列型や数値型など様々なデータの種類があり、データ型を呼ばれている。
  • データ型の違い→
    print(5 + 7)
    #結果｢12｣←数値の計算
    print("5" + "7")
    "#結果｢57｣←文字列の連結

  • 型変換str→
    price = 100
    print("りんごの価格は" + price + "円です")
    #結果エラー←文字型を数値型なので連結できない
    price =100
    print("りんごの価格は" + str(price) + "円です")
    #結果りんごの価格は100円です←文字型同士なので連結できる
    上のようにデータ型を変換することを「型変換」といい、数値型から文字列型に変換するには「str」を用いる。str=string（読み＝ストリング、意味＝一続き、一列。

３．変数の更新

  • 変数の値を更新1→
    #変数xを定義
    x=5
    print(x)「５」と出力
    x=11
    print(x)上書きされ「１１」と出力
    一度定義した変数の値を、「変数名=新しい値」とすることで上書きすることができる。
  • 変数の値を更新2→
    #変数xを定義
    x=3
    print(x)「３」と出力
    x=x+5
    print(x)上書きされ「８」と出力
    変数自身に数値を足したものを再び同じ変数に代入することで値を上書きできる。=は代入という意味を再確認。
  • 変数の値を更新3→
    ＃基本形
    x=x+10
    x=x*10
    ＃省略形
    x+=10
    x*=10
    数値の入った変数を更新する場合、省略することができる。よく使う形。

２．変数の定義、規格、意義

  • 変数をイメージ→変数とは、データ（値）を入れておく箱のようなもの。この箱（変数）に名前（変数名）をつけることでその名前を用いて変数に値を入れたり変数から値を取り出したりできる。
  • 変数の定義→変数名="数値"で定義する。数値のみクォーテーションで囲む。プログラミングの＝は右辺を左辺に代入するという意味。
  • 変数の値を抽出→
    name="john"
    print(name)
    とすると変数nameの値"john"が出力。print("name") としてしまうと文字列"name"を出力する。
  • 変数名のつけ方→英単語を用いる。２語以上の場合は＿で区切る。数字で開始、ローマ字、日本語は推奨されていないもしくはできない。
  • 変数を使う意義１→
    ☆使用した場合☆
    apple    _count=3
    apple_price=100
    toral_price=apple_count* apple_price
    print(total_price)結果出力
    ☆使用しない場合☆
    total__price=3*100
    print(total_price)結果出力
    のようにデータに名前を付けることで扱っているデータの中身が何を表しているか明確になる。その結果コードが読みやすい。変数を使うメリット。
  • 変数を使う意義２→
    ☆使用した場合☆
    #正方形の面積を計算
    length=5
    area=length*length
    print(area)結果出力
    ☆使用しない場合☆
    #正方形の面積を計算
    area=5*5
    print(area)結果出力
    のように同じデータを繰り返し使用することができ、変数の値に変更があった場合変更する箇所が１箇所ですむ。
